# indigo AI

A customizable AI chatbot with persona management, memory, and multi-modal capabilities.

## Features

- **Chat**: Conversational AI with customizable personality, powered by Gemini 3.1 Pro Preview.
- **Multi-modal Input**: Upload images and text files (knowledge base) to the chat.
- **AI Persona**: Customize the AI's name, personality, backstory, appearance, reference image, and voice.
- **User Profile**: Set your own name, background, and preferences.
- **Memory & Knowledge Base**: View indexed text files and conversation stats.
- **Gallery**: View and download uploaded and generated images.
- **Image Generator**: Generate images based on prompt, AI appearance, and reference image.
- **Reflective Journal**: AI generates daily reflections based on conversations.
- **Settings**: Export/Import data for backup, dark mode (UI only), and notifications.
- **Text-to-Speech & Speech-to-Text**: Voice interaction support.

## Technologies

- React 19
- Tailwind CSS v4 (Indigo Theme)
- Google GenAI SDK
- Lucide React Icons
- React Router DOM
